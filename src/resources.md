# Resources

**A collection of various helpful resources**

## Webmaster resources

[Sparrow's Guide to Accessible Webmastery](https://accessible-webmastery.neocities.org/) - Guides and resources to make your website more accessible!

[HTML Cheat Sheet](https://htmlcheatsheet.com/) - an HTML cheatsheet

[sadgrlonline](https://sadgrl.online/resources/) - webmaster resources from sadgrl

## Online reference libraries for artists
- [Line of Action](https://line-of-action.com/) - reference photos and practice tools for visual artists including figure drawing, animal drawing, scenes & environment drawing, and more. 
	- [Support the creator](https://line-of-action.com/membership)
- [Afro Index](https://afroindex.org/)- A Reference Library for Black Hairstyles
	- [Support the creator](https://afroindex.org/support)
- [Posemaniacs](https://www.posemaniacs.com/)- free 3d anatomy + pose reference library 
- [x6ud.github.io/#/](https://x6ud.github.io/#/)-  Find animal reference photos based on skull angle
	- [Support the creator here](https://ko-fi.com/x6udpngx)
- [x6ud.github.io/pose-search](https://x6ud.github.io/pose-search/#/)- Find pose reference photos based on poseable skeleton
	- [Support the creator here](https://ko-fi.com/x6udpngx)
- [FilmGrab](https://film-grab.com/)- large film reference library of movie stills 
	- [Support the creator here](https://www.patreon.com/filmgrab)

## Online resources and tools for artists
- [Coolors.co](https://coolors.co/)- color palette generator 
- [Lospec](https://lospec.com/)- tutorials, palette lists, gallery, tools and resources for mainly pixel art and other low-spec forms of art  
	- [Support the creator here](https://www.patreon.com/lospec/)
  
## Useful software for artists 

- [pureref](https://www.pureref.com/)- program available for Windows, Mac and Linux that helps view and organize reference images on your desktop. Free *if* you set the contribution price to $0 at checkout
	-  [Support the creators here](https://www.pureref.com/support_us.php)
